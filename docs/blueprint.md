# **App Name**: Brittany Chiang Reimagined

## Core Features:

- Profile Section: Display a brief introduction about Brittany Chiang, highlighting key skills and experiences.
- Projects Showcase: Present a selection of Brittany Chiang's notable projects, each with a title, brief description, and link to the live project or source code.
- Skills Section: Visually represent Brittany Chiang's technical skills, potentially using a bar graph or icon-based display for each skill.
- Theme selection: Switch between dark and light themes

## Style Guidelines:

- Primary color: Midnight Blue (#2C3E50) for a professional and sophisticated feel.
- Background color: Off-White (#F0F4F8) to provide a clean and modern backdrop.
- Accent color: Teal (#008080) to highlight interactive elements and key information.
- Body font: 'Inter', a sans-serif font, known for its clear legibility on screens.
- Headline font: 'Space Grotesk', a bold and modern sans-serif, providing clear hierarchy.
- Use a set of minimalist icons (Line Awesome) to represent skills, technologies, and links to social media.
- Implement a clean, single-page layout with clear sections and a focus on whitespace to enhance readability.
